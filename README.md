# amyy-css

Repository containing CSS files used across amyy's (that's me!) projects.

## How to Use

CSS is applied directly to the body and other HTML elements to make it work with
the [mkdocs theme](https://github.com/amyy54/mkdocs-amyy), so this CSS should
only be used on websites dedicated to it. Furthermore, all content should be
wrapped in a `<div class="wrap">` block to make the CSS play nice.

`main-center` is used to create a centered view that corresponds to the content
size, as shown on [amyy.me](https://amyy.me). `main-stretch` will stretch the
block to fit the entire screen, and will not center the text vertically, as
shown on [amyy.me/docs](https://amyy.me/docs).
